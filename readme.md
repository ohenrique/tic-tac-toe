# Tic tac toe
Tic-tac-toe (American English), noughts and crosses (British English), jogo da velha (Brazil :heart_eyes:), or Xs and Os is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Getting started
First, you should be using yarn, if you don't, start installing by
```npm install -g yarn```

Now, run ```yarn``` to restore the dependencies.

## Running
To start the application just run the command
```yarn start```

Or, just for specific platform
#### Android
```yarn android```

#### iOS
```yarn ios```

#### Web
```yarn web```

## Testing
Jest is already configured, run ```yarn test```
